This interactive web application demonstrates AI-powered customer service agents with distinct personalities built using TinyTroupe and Streamlit.

### Features

- **Two Unique AI Agents:**
  - **Sunny Martinez** - Enthusiastic, warm, and optimistic customer service representative
  - **Alex Chen** - Efficient, professional, and systematic customer service representative

- **Interactive Chat Interface** - Real-time conversations with AI agents
- **Personality-Driven Responses** - Each agent responds according to their unique personality traits
- **Professional UI** - Clean, user-friendly Streamlit interface

### Technologies Used

- **TinyTroupe** - AI agent simulation framework
- **Streamlit** - Web application framework
- **OpenAI GPT** - Large language model for natural conversations
- **Python** - Core programming language

### Agent Specifications

The agents are defined in JSON files with detailed personality profiles including:
- Personality traits (Big Five dimensions)
- Communication styles
- Professional backgrounds
- Behavioral patterns
- Skills and expertise
